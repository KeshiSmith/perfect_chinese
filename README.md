# # 完美中文
完美中文是一款用于汉化 Blender 插件的插件。
#### # 目前完成的功能
- [x] 全局翻译: 完善部分 Blender 的翻译
- [x] 一键翻译: 一键切换中英文的快捷按钮
- [x] 插件汉化: 目标插件的汉化
- [x] 更新插件: 检查和更新当前插件
- [x] 界面搜索：插件汉化界面的搜索功能
#### # 正在开发中的功能
- [ ] 开发者组件：一键机翻目标插件
#### # 未来要实现的功能
- [ ] 侧边栏：便捷的插件入口
